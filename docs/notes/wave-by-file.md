# Параллельные агенты — строго по непересекающимся файлам, крейт всегда компилируется

Перед запуском волны билдеров: (1) каждому агенту — свой кластер файлов, чужое не трогать;
(2) новые модули скаффолдить ДО волны (пустой компилирующийся каркас + mod-строка в lib.rs),
иначе у параллельных агентов падает cargo на «чужом» файле; (3) в промпт каждому — правило
«ошибки компиляции вне твоего кластера → подожди 60–90с и повтори, не чини чужие файлы»;
(4) конкурентные cargo-вызовы сериализуются file-lock'ом target/ — это норма, не зависание.
Константы, на которые завязаны соседи (PARAKEET_FILES и т.п.), кладём в каркас сразу.
