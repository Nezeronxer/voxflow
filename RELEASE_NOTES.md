# VoxFlow v2.0.9 — чёткий финал в кружке, надёжное обновление, чище главный экран

Patch-релиз поверх 2.0.8. Точность распознавания и синхронизацию кружка чинили
в 2.0.7–2.0.8; здесь — три доработки по обратной связи: финальный текст в Flow Bar
теперь видно достаточно долго, установщик обновления на Windows больше не зависает,
а на главном экране убрана дублирующая кнопка диктовки. Модели прежние —
GigaAM v3 (русский) и Parakeet TDT v3 (английский).

## Кружок держит финал заметно дольше

- Финальный, уже вставленный текст остаётся в плашке 1500 мс вместо 360 мс.
  Раньше итог лишь мелькал — казалось, что «в кружке одно, а в поле другое».
  Теперь результат видно и можно сверить со сказанным. Живой черновик и раньше
  собирался тем же локальным движком, что и финал (см. 2.0.7–2.0.8).

## Обновление на Windows больше не зависает

- В `install_update` добавлен гарантированный форс-выход: если штатный
  `app.exit(0)` застревает на потоке/обработчике, процесс завершается жёстко через
  1.5 с, чтобы установщик мог заменить работающий exe. Это устраняет ситуацию
  «скачалось, но ничего не происходит, приложение зависло».

## Меньше лишнего на главном экране

- Убрана дублирующая кнопка «Начать диктовку»: большой микрофон-орб и строка
  статуса запускают и останавливают запись без второй кнопки под ними.

## Проверка

- 21 frontend-тест пройден.
- 299 Rust-тестов пройдено (5 model/audio/platform пропущены).
- `check_versions.py --tag v2.0.9` — версии консистентны.
- Windows x64 и macOS ARM64 собираются и публикуются одним release workflow.
