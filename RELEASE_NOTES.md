# VoxFlow v2.0.5 — точнее, быстрее и с памятью исправлений

VoxFlow 2.0.5 делает локальную диктовку предсказуемее: итог в поле совпадает с
проверенным финальным текстом, случайные слова на тишине отбрасываются, а ручные
исправления пользователя начинают помогать следующим распознаваниям.

## Распознавание и контекст

- Автоматический язык снова использует Whisper Large v3 Turbo как основной
  многоязычный маршрут. Parakeet остаётся быстрым специализированным вариантом
  для явно выбранного английского, GigaAM — для русского.
- Добавлены консервативные фильтры сегментов по вероятности отсутствия речи и
  качеству декодирования. Это уменьшает фантомные слова на тишине и шуме.
- Словарь, голосовые триггеры сниппетов, хвост текущего поля и уже выученные
  исправления входят в ограниченную ASR-подсказку без содержимого сниппетов.
- Автоматический AI-рефайн не может подменить исходную мысль несвязанным текстом;
  итоговая вставка проходит проверку привязки к распознанной фразе.
- Плавающая плашка сохраняет ровно тот финальный текст, который будет вставлен,
  без второй скрытой замены после отображения.

## Память ручных исправлений

- После собственной вставки VoxFlow на короткое время отслеживает только факт
  ручного редактирования и затем перечитывает то же активное текстовое поле через
  UI Automation / Accessibility. Нажатые символы глобально не записываются.
- Запоминаются только связанные замены слов и коротких фраз. Допечатывание нового
  текста, смена окна/поля, устаревшая правка, парольное поле и похожие на секреты
  значения отбрасываются.
- Повторная правка усиливает одно правило без Unicode-дубликатов; новое явное
  исправление заменяет старое неоднозначное правило.

## Словарь и сниппеты

- Ошибки сохранения больше не скрываются интерфейсом; состояние обновляется
  только после подтверждённой записи в SQLite.
- Пустая замена в словаре означает «предпочитать это точное написание», а не
  удалять слово. Повторные записи объединяются.
- Slash-сниппет `/адрес` срабатывает на «адрес», «слэш адрес» и «сниппет адрес»
  только как отдельная команда. Шаблоны понимают русские и английские переменные
  даты, времени и буфера обмена.
- Каталог моделей сокращён до сильных и реально используемых вариантов; основной
  рекомендованный выбор — Whisper Large v3 Turbo.

## Скорость и стабильность

- Опциональные live-preview больше не ждут занятый ASR/model mutex и не блокируют
  команды старта, остановки и отмены.
- Прогрев и тяжёлые fallback-пути выполняются вне командного цикла; устаревшее
  поколение диктовки проверяется до дорогого повторного распознавания.
- На macOS сокращены лишние Accessibility-опросы, а frontmost-приложение берётся
  через NSWorkspace с безопасным AX fallback. Это устраняет ложную потерю цели в
  Chrome/ChatGPT и уменьшает задержку перед вставкой.

## macOS

- Релиз собирает отдельные ARM64 `.dmg` и `.app.zip` для Apple Silicon/macOS 11+.
- Без Apple Developer secrets артефакты полностью проверяются и запечатываются
  ad-hoc подписью. При первом запуске macOS может потребовать «Всё равно открыть».
- Pipeline готов к Developer ID signing и notarization через одноразовый keychain,
  когда владелец добавит полный набор Apple secrets; неполная конфигурация
  останавливает публикацию вместо выдачи неверно помеченного артефакта.
