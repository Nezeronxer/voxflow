# VoxFlow — локальный голосовой ввод

> Жмёшь горячую клавишу → говоришь по-русски → чистый текст с пунктуацией вставляется в **активное поле любого приложения**. Полностью локально и офлайн. Аналог Wispr Flow, но приватный и без облака.

![status](https://img.shields.io/badge/platform-Windows-blue) ![offline](https://img.shields.io/badge/offline-yes-green) ![lang](https://img.shields.io/badge/язык-RU%20first-orange)

## Что это

VoxFlow — десктоп-приложение голосового ввода на **Tauri 2** (Rust + React). Распознавание речи — локальный **whisper.cpp** (модель `large-v3-turbo`), сильный по русскому. Никакие аудио и тексты не уходят в облако.

- 🎙️ **Hold-to-talk** (зажми клавишу и говори) и **toggle** (вкл/выкл).
- 💉 **Off-focus вставка** — текст попадает в то окно, что было активно (Telegram, браузер, Word, IDE…), VoxFlow остаётся в фоне.
- ✍️ **Авто-пунктуация и капитализация** без проговаривания знаков.
- 🧹 **Удаление слов-паразитов** («эм», «ну», «как бы», «типа», «короче»…), настраивается.
- 📖 **Личный словарь** (имена, термины, бренды) — биасинг + замены.
- ⚡ **Сниппеты / горячие слова** — фраза-триггер разворачивается в шаблон (с переменными `{date}`, `{time}`, `{clipboard}`).
- 🎚️ **Verbatim-режим** — точная расшифровка без «улучшений».
- 🌍 **Мультиязык** (auto-detect), приоритет — русский.
- 🔒 **Полный офлайн** после скачивания модели.
- 📊 Статистика: слова за день/всего, сессии, серия дней.

## Установка (Windows)

1. Скачайте и запустите инсталлятор **`VoxFlow_0.1.0_x64-setup.exe`**
   (из `voxflow/src-tauri/target/release/bundle/nsis/`).
2. Установка идёт **в профиль пользователя** (без прав администратора).
3. Запустите VoxFlow — он появится в **системном трее** (рядом с часами).
4. При первом запуске откройте **Настройки → Модель** и нажмите «Скачать» у
   *Large v3 Turbo Q5* (≈574 МБ, рекомендуется). Модель качается один раз и
   дальше всё работает офлайн.
   - Если нужен быстрый старт на слабом ПК — скачайте *Base* (148 МБ).

> Нужен установленный **WebView2 Runtime** (есть на всех актуальных Windows 10/11; инсталлятор подтянет при отсутствии).

## Как пользоваться

1. Поставьте курсор в любое текстовое поле (чат, документ, адресная строка).
2. **Зажмите правый `Ctrl`** (по умолчанию) и говорите.
3. Отпустите — через секунду в поле появится распознанный текст с пунктуацией.

Внизу экрана во время записи всплывает индикатор «Слушаю… / Распознаю…».

### Горячая клавиша
- По умолчанию — **правый Ctrl** (hold-to-talk). Меняется в *Настройки → Управление*.
- Режим **toggle**: одно нажатие — старт, второе — стоп.
- ⚠️ **Про Fn:** на большинстве Windows-клавиатур клавиша `Fn` обрабатывается
  прошивкой и **не видна ОС** — поэтому по умолчанию не `Fn`. Если ваша `Fn`
  перехватывается — её можно назначить, но это не гарантируется железом.

## Настройки (вкладки)

| Вкладка | Что внутри |
|---|---|
| **Главная** | статус, текущий хоткей, статистика, кнопка диктовки, последняя расшифровка |
| **Модель** | выбор/скачивание модели whisper, язык, движок, число потоков |
| **Распознавание** | verbatim, удаление паразитов, авто-пунктуация, тон, способ вставки |
| **Управление** | устройство ввода, горячая клавиша, режим, звуки, автозапуск |
| **Словарь** | термины и замены (биасинг распознавания) |
| **Сниппеты** | триггер → текст/шаблон |
| **История** | последние диктовки |

**Способ вставки:** «Вставка» (через буфер обмена — надёжно, быстро; буфер
восстанавливается) или «Печать» (посимвольно). По умолчанию — вставка.

## Приватность

- Распознавание — **локально**, на вашем CPU. Аудио никуда не отправляется.
- Постобработка (паразиты/пунктуация/тон по умолчанию) — **локальная**,
  детерминированная, без LLM и без сети.
- Модель скачивается с HuggingFace **один раз**; дальше интернет не нужен.
- Автозапуск — **только если вы сами включите** тумблер.

## Сборка из исходников (для разработчиков)

Требуется: **Rust** (stable, MSVC toolchain), **Node 18+**, **Visual Studio Build
Tools** (MSVC). cmake/clang НЕ нужны — whisper.cpp используется как готовый
sidecar-бинарь (лежит в `voxflow/src-tauri/resources/whisper/Release/`).

```bash
cd voxflow
npm install
npm run tauri dev      # запуск в режиме разработки
npm run tauri build    # сборка инсталлятора (NSIS) в src-tauri/target/release/bundle/nsis/
```

Диагностика пайплайна без GUI (прогон ASR+постобработки на 16 кГц WAV):
```bash
voxflow/src-tauri/target/debug/voxflow.exe --selftest C:/путь/к/audio_16k.wav
```

### Архитектура (кратко)
- **Оболочка:** Tauri 2 (Rust) + React 19 / TypeScript / Vite.
- **ASR:** whisper.cpp v1.8.6 sidecar (`whisper-cli.exe`), модель GGML.
- **Аудио:** `cpal` (захват) + ресемпл 16 кГц + энергетический VAD + `hound` (WAV).
- **Хоткей:** `rdev` (low-level hook) — honest hold-to-talk на любой клавише.
- **Инжект:** `arboard` (буфер) + `enigo` (Ctrl+V / печать).
- **Хранилище:** SQLite (`rusqlite`, bundled) — настройки, словарь, сниппеты, история, статистика.
- **Загрузка моделей:** системный `curl` (без тяжёлого HTTP-стека).

Все архитектурные решения и их обоснования — в [`DECISIONS.md`](DECISIONS.md).
Граф задач — [`WORKFLOW.md`](WORKFLOW.md). Что реально проверено — [`PROGRESS.md`](PROGRESS.md).
